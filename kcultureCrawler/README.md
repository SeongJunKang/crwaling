- python version : 3.6.7
- selenium (chrome Driver)
- windows 10

## contents
- diction으로 저장된 파일은 [한식문화사전]의 크롤링 소스입니다.<br>
- sympathy로 저장된 파일은 [한식문화공감]의 크롤링 소스입니다. (URL을 각각 입력해주어야합니다)<br>
